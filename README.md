# lpms_me1_sample
Sample code for LPMS-ME1 IMU sensor
## 開発環境

|    | Description |
| --- | --- |
| CPU | STM32f303k8 |
| IDE | STM32CubeIDE 1.3.0 |
| Sensor kit | LPMS-ME1-DK Ver.1.2 |
